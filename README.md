# File Server v1.0

Современный веб-файловый сервер на Flask с личными кабинетами, общими папками и удобным интерфейсом в стиле настольного проводника.

## Установка

``` bash
git clone https://github.com/Lentayi/file-server.git
cd file-server
pip install -r requirements.txt
```

## Если admin не создан
Запустите скрипт 
``` bash
python create_admin.py
```



## Возможности

- авторизация пользователей
- личное хранилище для каждого пользователя
- общие папки с настройкой доступа
- загрузка, создание папок, переименование, копирование, вырезание, вставка и удаление файлов
- отдельный просмотрщик всех файлов для администратора
- управление пользователями через админ-панель
- выдача ролей и прав доступа
- смена пароля и аватара
- светлая и темная тема
- автоматическое переключение темы по времени суток
- аккуратный дашборд с информацией о хранилище и быстрым переходом к разделам

## Роли

### Пользователь
- доступ к личному хранилищу
- просмотр доступных общих папок
- смена пароля
- смена аватара
- просмотр своей роли и прав

### Администратор
- все возможности обычного пользователя
- создание новых пользователей
- выдача прав администратора
- управление доступом к общим папкам
- создание и изменение общих папок
- просмотр всех файлов и папок на сервере, включая личные хранилища пользователей
